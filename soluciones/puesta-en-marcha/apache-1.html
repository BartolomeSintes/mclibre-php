Se ofrecen a continuación unas posibles soluciones de los ejercicios Apache.

Apache (1) 1 - Creación de alias

  • Incluya el siguiente alias en httpd.conf 
    <IfModule alias_module>
# ...
    # 2021-09-16. Ejercicios Apache. Ejercicio 1
    Alias /ejercicio-01 "C:\tmp\apache-1\e01"
    <Directory "C:\tmp\apache-1\e01">
      Options Indexes FollowSymLinks Includes ExecCGI
      AllowOverride All
      Require all granted
    </Directory>
# ...
</IfModule>

Apache (1) 3 - Creación de área de trabajo (carpeta), tareas y alias

  • Incluya las siguientes tareas en tasks.json 
        {
      "label": "Abrir en localhost (Firefox)",
      "command": "explorer",
      "windows": {
        "command": "C:\\Program Files\\Mozilla Firefox\\firefox.exe"
      },
      "args": [
        "localhost/ejercicio-03/${relativeFile}"
      ],
      "presentation": {
        "reveal": "never"
      }
    },
    {
      "label": "Abrir en localhost (Chrome)",
      "command": "explorer",
      "windows": {
        "command": "C:\\Program Files\\Google\\Chrome\\Application\\chrome.exe"
      },
      "args": [
        "localhost/ejercicio-03/${relativeFile}"
      ],
      "presentation": {
        "reveal": "never"

    }
