Se ofrecen a continuación unas posibles soluciones de los ejercicios Apache y Eclipse.

Apache 1 - Creación de alias

  • Cree la carpeta C:\tmp\apache-eclipse\e01
  • Incluya el siguiente alias en httpd.conf 
    <IfModule alias_module>
# ...
    # 2018-09-17. Ejercicios Apache-Eclipse. Ejercicio 1
    Alias /ejercicio-01 "C:\tmp\apache-eclipse\e01"
    <Directory "C:\tmp\apache-eclipse\e01">
      Options Indexes FollowSymLinks Includes ExecCGI
      AllowOverride All
      Require all granted
    </Directory>
# ...
</IfModule>
  • Reinicie Apache
