Introducción

Esta lección contiene un índice de las lecciones que estoy impartiendo en clase durante el primer trimestre del curso 2021/22 (y parte del segundo), así como los ejercicios que han realizado los alumnos.

Para elaborar este índice utilizo como base las guías de estudio que ofrezco a mis alumnos de la modalidad semipresencial en la plataforma Moodle de la Conselleria de Educación. El trabajo propuesto a estos alumnos es el mismo que siguen en clase mis alumnos de la modalidad presencial, aunque la distribución temporal no sea exactamente la misma.

La información se ofrece distribuida en semanas, ya que es así como está organizada la modalidad semipresencial.

También se pueden consultar las lecciones impartidas en el pasado curso 2019/20 (en el curso 2020/21 no impartí este curso).

IAW Semana S01: del 13/09/21 al 19/09/21

La primera parte del curso estará dedicada a la programación en PHP. Los apuntes y ejercicios de esta parte están disponibles en la web: https://www.mclibre.org/consultar/php

La página principal de los apuntes da acceso a todas las lecciones (que iré modificando a lo largo del curso). Cada semana os pondré aquí enlaces a las lecciones que quiero que leáis y a los ejercicios que quiero que hagáis.

Estas son las actividades a realizar durante esta semana.

Presentación del curso Programación web en PHP

  1. Leed la lección Presentación de los apuntes.

¿Qué es PHP?

  1. Leed la lección Lenguajes de programación.
  2. Leed la lección Qué es PHP.

Instalación de XAMPP

  1. Leed la lección Qué es XAMPP.
  2. Leed la lección Instalación de XAMPP en Windows.
  3. Descargad XAMPP 8.0.10 (161 MB).
  4. Instalad XAMPP 8.0.10 de acuerdo con la lección Instalación de XAMPP en Windows.

    Nota: Una vez instalado XAMPP, haced una copia de seguridad de la carpeta /xampp/apache/conf/ y del archivo /xampp/php/php.ini por si necesitáis recuperarlos si cometéis errores al configurar XAMPP.

Configuración de Apache y PHP

  1. Leed la lección Configuración de Apache y PHP.

    En esta lección se tratan dos puntos muy importantes:

    1. los Alias de Apache, que utilizaremos durante todo el curso.
    2. las opciones de configuración de PHP, que deben modificarse como se indica en la lección para que el comportamiento sea el mismo en los ordenadores de todos los alumnos.
  2. Configurad php.ini de acuerdo con la lección Configuración de Apache y PHP.
  3. Para familiarizaros con la creación de Alias en Apache, haced los ejercicios Apache 1 y 2.
  4. En la página de soluciones podéis consultar soluciones de los ejercicios.

Visual Studio Code

Este curso vamos a utilizar Visual Studio Code (VSCode) como editor para PHP. A lo largo del curso iré completando y ampliando estas lecciones dedicadas a Visual Studio Code

  • Lecciones en los apuntes de Informática:
    1. Leed la lección Introducción.
    2. Leed la lección Instalación.

      En esta lección se explica cómo conseguir e instalar VSC.

    3. Descargad Visual Studio Code (64 bits, recomendado)
    4. Instalad Visual Studio Code.
    5. Leed la lección Personalización.

      En esta lección os explico cómo están organizadas las posibilidades de personalización de VSC.

    6. Leed la lección Uso.

      En esta lección iré añadiendo explicaciones del manejo de Visual Studio Code a medida que avancemos en su uso.

  • Lecciones en los apuntes de HTML/CSS:
    1. Leed la lección Configuración recomendada para HTML/CSS.

      En esta lección os detallo la configuración de VSC que quiero que utilicéis.

    2. Configurad Visual Studio Code de acuerdo con la lección Configuración recomendada para HTML/CSS.
    3. Leed la lección Uso de la extensiones recomendadas para HTML/CSS.
  • Lecciones en los apuntes de PHP:
    1. Leed la lección Configuración recomendada para PHP.

      En esta lección os detallo la configuración de VSC que quiero que utilicéis.

    2. Completad la configuración de Visual Studio Code de acuerdo con la lección Configuración recomendada para PHP.

Primeras páginas en PHP

  1. Leed la lección Primeras páginas en PHP
  2. Leed la lección Cadenas

    El estilo que se recomienda en esta lección es el que tenéis que utilizar en vuestros programas.

En la primera parte del curso de PHP vamos a hacer un pequeño repaso general de PHP, tocando los temas vistos el año pasado en el módulo LMSGI: variables, operaciones, if ... else .. y for ... además de las matrices, un tipo de variable que permite almacenar varios datos distintos en una sola variable. Esta parte la he llamado PHP, sin formularios porque las páginas no contienen formularios, por lo que el usuario no puede proporcionar información a los programas. Para que cada vez que se ejecuten los programas los resultados sean distintos, recurriremos a números aleatorios.

Variables

  1. Leed la lección Variables 1

    Esta semana no se realizan ejercicios con variables, pero quiero que leáis esa primera lección.

  2. Haced los ejercicios 4 a 8 de Primeras páginas.

    En estos ejercicios se necesita la función rand() que genera números aleatorios que se comenta en la lección Operaciones aritméticas (apartado Números aleatorios).

  3. En la página de soluciones podéis consultar soluciones de los ejercicios.
