PHP Intellisense

En julio de 2021 el autor retiró esta extensión del Marketplace, no tengo claro por qué, parece que porque no le puede dedicar tiempo. Los comentarios en las issues son que la gente está utilizando PHP Intelephense, que tiene versión gratuita y de pago. Así que en septiembre de 2021 la he quitado de los apuntes.

Autor: Felix Becker

Marketplace: PHP Intellisense

Repositorio GitHub: PHP Intellisense

PHP Intellisense proporciona autocompletado, argumentos de funciones, acceso directo a definiciones, etc..

Preferencias globales recomendadas para PHP Intellisense:

Nota: Estas preferencias están incluidas en el apartado anterior de preferencias de configuración. No es necesario añadirlas si ya se han añadido anteriormente.

Tenga en cuenta que puede ser necesario modificar la preferencia resaltada del ejemplo siguiente:

  • "php.executablePath": debe indicar la ruta hasta el ejecutable de PHP. El valor incluido en el ejemplo es el predeterminado de XAMPP en Windows.
  • La ruta puede escribirse con barras (/, como en el ejemplo) o contrabarras dobles (\\), pero no contrabarras simples (\).
  // Extensión PHP Intellisense
  //
  // PHP: Executable Path
  // Ubicación del intérprete de PHP.
  "php.executablePath": "C:/xampp/php/php.exe",
  //
  //

PHP Open PHP/HTML/JS In Browser

Esta extensión se puede usar como alternativa a la creación de tareas y de atajos de teclado para abrir las páginas en los navegadores. La he descartado para el curso 2019/20 porque esta extensión sólo permite un atajo de teclado (Shift + F6). He preferido que los alumnos definan varios atajos de teclado y puedan elegir con atajos cómo se abren las páginas (como ficheros o en el servidor) y con qué navegador (Firefox o Chrome).

Autor: PrimaFuture

Marketplace: Open PHP/HTML/JS In Browser

Repositorio GitHub: No disponible

Preferencias de área de trabajo o carpeta

{
  // Configuración de extensiones recomendadas para el curso Programación web en PHP de www.mclibre.org
  //
  // Extensión Open PHP/HTML/JS in Browser
  //
  // Open PHP/HTML/JS in Browser: URL to open
  // Tipo de URL a abrir en el navegador (custom = personalizada)
  "open-php-html-js-in-browser.urlToOpen": "custom",
  //
  // Open PHP/HTML/JS in Browser: Document Root Folder
  // Directorio raíz de las páginas web
  "open-php-html-js-in-browser.documentRootFolder": "C:/Users/NombreDeUsuario/Documents/IAW/PHP",
  //
  // Open PHP/HTML/JS in Browser: Custom URL to open
  // Plantilla de URL a abrir en el navegador
  "open-php-html-js-in-browser.customUrlToOpen": "http://localhost/iaw/php/${relativeFile}",
  //
  // Open PHP/HTML/JS in Browser: Browser to open
  // Navegador a abrir
  "open-php-html-js-in-browser.selectedBrowser": "Firefox",
  //
}
