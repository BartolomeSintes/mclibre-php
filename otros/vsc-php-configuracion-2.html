Otras extensiones para el curso de PHP

PHP Open PHP/HTML/JS In Browser

Esta extensión se puede usar como alternativa a la creación de tareas y de atajos de teclado para abrir las páginas en los navegadores. La he descartado para el curso 2019/20 porque esta extensión sólo permite un atajo de teclado (Shift + F6). He preferido que los alumnos definan varios atajos de teclado y puedan elegir con atajos cómo se abren las páginas (como ficheros o en el servidor) y con qué navegador (Firefox o Chrome).

Autor: PrimaFuture

Marketplace: Open PHP/HTML/JS In Browser

Repositorio GitHub: No disponible

Preferencias de área de trabajo o carpeta (Área de trabajo)

{
  // Configuración de extensiones recomendadas para el curso Programación web en PHP de www.mclibre.org
  //
  // Extensión Open PHP/HTML/JS in Browser
  //
  // Open PHP/HTML/JS in Browser: URL to open
  // Tipo de URL a abrir en el navegador (custom = personalizada)
  "open-php-html-js-in-browser.urlToOpen": "custom",
  //
  // Open PHP/HTML/JS in Browser: Document Root Folder
  // Directorio raíz de las páginas web
  "open-php-html-js-in-browser.documentRootFolder": "C:/Users/NombreDeUsuario/Documents/IAW/PHP",
  //
  // Open PHP/HTML/JS in Browser: Custom URL to open
  // Plantilla de URL a abrir en el navegador
  "open-php-html-js-in-browser.customUrlToOpen": "http://localhost/iaw/php/${relativeFile}",
  //
  // Open PHP/HTML/JS in Browser: Browser to open
  // Navegador a abrir
  "open-php-html-js-in-browser.selectedBrowser": "Firefox",
  //
}
