Comprobación instalación

Para comprobar que se han configurado Apache y PHP de acuerdo con la página de configuración se puede utilizar la función phpinfo($tipo_de_informacion), que muestra gran cantidad de información acerca de la configuración de Apache y PHP.

Para ello, es suficiente crear un archivo PHP con el siguiente contenido:

<?php
print "<h1>Comprobación de la instalación</h1>\n";
phpinfo();
?>

y ejecutarlo en el navegador.

Al ejecutar este archivo se mostrará una página en la que hay que buscar y comprobar los siguientes valores:

Configuración de Apache y de PHP
Variable Valor
DOCUMENT_ROOT C:/xampp/htdocs
short_open_tag Off
error_reporting 32767 (E_ALL)
PDO drivers mysql, sqlite
