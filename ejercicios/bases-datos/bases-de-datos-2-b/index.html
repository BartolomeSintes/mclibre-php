Bases de datos (2 B)

Enunciados

  • 0 - Aplicación de ejemplo
  • 1 - Añadir campo
  • 2 - Añadir campo

Para facilitar la realización de los ejercicios, se proporcionan unas plantillas de los ejercicios Bases de datos (2 B) (Bases de datos):

  • La aplicación de ejemplo es como la solución del ejercicio Bases de datos (2) 2. La plantilla del ejercicio 1 es como la aplicación de ejemplo. La plantilla del ejercicio 2 es como la solución del ejercicio 1.
  • Las plantillas incluyen tres ficheros .side que permiten la comprobación con Selenium IDE del funcionamiento de la aplicación de ejemplo y de los ejercicios.

Puede descargar unas posibles soluciones completas de estos ejercicios.

Bases de datos (2 B) - 0 - Aplicación de ejemplo

Esta aplicación es la aplicación que se puede utilizar de base para el resto de ejercicios (es idéntica al ejercicio Bases de datos (2) 2).

Bases de datos (2 B) - 1 - Añadir campo

Añada a la aplicación de ejemplo anterior un tercer campo para guardar el número de teléfono. El campo puede tener 15 caracteres de longitud, de acuerdo con la norma ITU-T E.164.

Bases de datos (2 B) - 2 - Añadir campo

Añada a la aplicación del ejercicio anterior un cuarto campo para guardar el correo electrónico (de longitud 50 caracteres).
