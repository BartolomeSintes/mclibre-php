Enunciados

  • 0 - Aplicación de ejemplo
  • 1 - Añadir campo

Para facilitar la realización de los ejercicios, se proporcionan unas plantillas de los ejercicios Bases de datos (1 B) (Bases de datos):

  • La aplicación de ejemplo es como la solución del ejercicio Bases de datos (1) 6. La plantilla del ejercicio es como la aplicación de ejemplo 0. La plantilla del ejercicio 2 es como la solución del ejercicio 1.
  • Las plantillas incluyen tres ficheros .side que permiten la comprobación con Selenium IDE del funcionamiento de la aplicación de ejemplo y de los ejercicios.

Puede descargar unas posibles soluciones de estos ejercicios en la página de soluciones.

Bases de datos (1 B) - 0 - Aplicación de ejemplo

Esta aplicación es la aplicación que se puede utilizar de base para el resto de ejercicios (es idéntica al ejercicio Bases de datos (1) 6).

Bases de datos (1 B) - 1 - Añadir campo

Añada a la aplicación de ejemplo anterior un tercer campo para guardar el número de teléfono (de longitud 10 caracteres).

Bases de datos (1 B) - 2 - Añadir campo

Añada a la aplicación del ejercicio anterior un cuarto campo para guardar el correo electrónico (de longitud 50 caracteres).
