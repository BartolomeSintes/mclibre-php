Memorión 1

Escriba un programa que muestre un número aleatorio de imágenes de animales que cambien cada vez que se actualice la página.

  • El número de imágenes estará entre 2 y 20.
  • Las imágenes serán todas distintas.
  • Las imágenes serán entidades numéricas, del rango entre 128000 (&#128000; = 🐀) y 28060 (&#128060; = 🐼).
  • Este programa no necesita utilziar sesiones.

Memorión 2

Añada al programa anterior dos páginas para permitir al usuario modificar el número de imágenes.

  • La primera vez que se cargue la página principal, se mostrarán 10 imágenes.
  • La página principal contendrán un enlace al formulario de configuración.
  • El formulario mostrará en el control el número actual de imágenes.
  • El número de imágenes se podrá elegir entre 2 y 61.
  • Si se envían valores incorrectos, se redirigirá a la primera página, sin necesidad de indicar nada al usuario.
  • Si la primera vez se intenta acceder a la segunda o tercera página directamente, se redirigirá a la primera página, sin necesidad de indicar nada al usuario.

Memorión 3

Modifique el programa anterior de manera que:

  • Cada imagen se muestre dos veces.
  • Las imágenes se muestren barajadas.
  • Las imágenes se guarden en una variable de sesión.

Memorión 4

Modifique el programa anterior de manera que:

  • Cada imagen se muestre dos veces.
  • Las imágenes se muestren barajadas.
  • Las imágenes se guarden en una variable de sesión.
