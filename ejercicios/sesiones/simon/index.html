Para facilitar la realización de los ejercicios, se proporcionan unas plantillas de los ejercicios Simon (Sesiones). Sólo se incluyen las plantillas para los tres primeros ejercicios, ya que se recomienda hacer cada ejercicio a partir del anterior.

Puede descargar unas posibles soluciones de estos ejercicios en la página de soluciones.

Simon 1 - Crear lista de colores

Escriba un programa de dos páginas que muestre una lista de colores.

  • La primera página contiene un formulario con cinco botones de tipo submit con el mismo name, cuatro para indicar el color elegido y uno para reiniciar el juego.
  • La segunda página recibe el dato, añade un color o reinicia y redirige a la primera página.

Simon 2 - Comprobar lista de colores

Amplíe el programa anterior:

  • Que muestre una lista de entre 2 y 7 colores, al azar.
  • Que indique si la lista de colores creada por el usuario coincide con la propuesta. En cuanto el usuario se equivoque, el juego termina.

Simon 3 - Separar listas de colores

Amplíe el programa anterior dividiéndolo en tres páginas:

  • En la primera página se muestre la lista de colores. Para pasar a la segunda página y empezar el juego, se debe hacer clic en un botón.
  • En la segunda página el usuario crea la lista de colores y se indica si se ha equivocado o ha completado la lista correcta.

Simon 4 - Aumentar la lista gradualmente

Amplíe el programa anterior:

  • La primera vez que se juegue, la lista de colores tendrá tres colores.
  • Cada vez que se consiga reproducir la lista de colores, el programa propondrá una lista más larga (un color más cada vez).

Simon 5 - Aumentar las opciones

Amplíe el programa anterior:

  • Indique el número de pasos que contiene la secuencia.
  • Si se falla, que el programa ofrezca la posibilidad de repetir la misma secuencia, además de volver a empezar desde el principio.
  • Si se falla, que el programa ofrezca la posibilidad de repetir una secuencia distinta de la misma longitud, además de pasar al nivel siguiente.
