Tragaperras 1

Escriba un programa que muestre tres imágenes de frutas. Al hacer clic en el botón Jugar se cambian las imágenes.

  • El formulario contiene un botón Enviar que no envía ningún dato y que representa que se juega. 
    print "      <td style=\"vertical-align: top; text-align: center\">"
    . "<p><input type=\"submit\" value=\"Jugar\" /></p></td>\n";
  • El formulario llama a la misma página. 
    print "<form action=\"$_SERVER[PHP_SELF]\" method=\"get\">\n\n";
  • Se generan aleatoriamente tres variables con números del 1 al 6 (para mostrar las imágenes) 
    $a = rand(1, 6);
$b = rand(1, 6);
$c = rand(1, 6);
  • Cada vez que se juega, cambia la combinación de frutas

Tragaperras 2

Añada al programa anterior la capacidad de indicar la apuesta.

  • Añada un botón de enviar que represente el haber metido una moneda en la máquina tragaperras. 
    print "        <p><input type=\"submit\" name=\"moneda\" value=\"Meter moneda\" /></p>\n";
  • Cada vez que se mete una moneda o se juega, se renueva la combinación de frutas.
  • Cada vez que se mete una moneda, la cantidad apostada aumenta en 1. 
    if ($moneda != "") {
    $apuesta += 1;
}
  • Muestre un contador de las monedas introducidas. 
    print "        <p style=\"margin: 0; font-size: 300%; border: black 4px solid; padding: 2px\">$apuesta</p>\n";
  • Al hacer clic en el botón Jugar, la cantidad apostada se debe poner a cero. 
    if ($jugar != "") {
    $apuesta = 0;
}
  • Incluya un control oculto con la cantidad apostada. 
    print "<p><input type=\"hidden\" name=\"apuesta\" value=\"$apuesta\" /></p>\n\n";
  • La validación de la cantidad apostada comprobará que se trata de un número entero mayor o igual que cero. En caso de no ser correcto, se le dará el valor 0. 
    if ($apuesta == "" || !is_numeric($apuesta) || !ctype_digit($apuesta)
    || $apuesta < 0) {
    $apuesta = 0;
}

Tragaperras 3

Añada al programa anterior la capacidad de conservar la última jugada en pantalla cuando se mete una moneda.

  • Sólo se renueva la combinación de frutas cuando se juega, no cuando se mete una moneda 
    if ($jugar != "") {
    $a = rand(1, 6);
    $b = rand(1, 6);
    $c = rand(1, 6);
}
  • Añada tres controles ocultos con los números de las tres imágenes que se están mostrando. 
    print "  <input type=\"hidden\" name=\"a\" value=\"$a\" />\n";
print "  <input type=\"hidden\" name=\"b\" value=\"$b\" />\n";
print "  <input type=\"hidden\" name=\"c\" value=\"$c\" /></p>\n\n";
  • La validación de los números de las imágenes debe comprobar que se encuentran en el rango de valores (entre 1 y 6). En caso de no ser correctos, se generará una nueva combinación. 
    $abcOk = true;
if ($a == "" || !is_numeric($a) || !ctype_digit($a) || $a < 1 || $a > 6simbolosNumer ||
    $b == "" || !is_numeric($b) || !ctype_digit($b) || $b < 1 || $b > 6 ||
    $c == "" || !is_numeric($c) || !ctype_digit($c) || $c < 1 || $c > 6) {
    $abcOk = false;
}

if (!$abcOk) {
    $a = rand(1, 6);
    $b = rand(1, 6);
    $c = rand(1, 6);

Tragaperras 4

Añada al programa anterior la capacidad de detectar que se ha ganado un premio.

  • Un orden en que los casos más especiales se escriben antes que los generales, lo que permite simplificar las condiciones, sería:
    • Si salen tres cerezas, se devuelve diez veces la apuesta.
    • Si salen dos cerezas, se devuelve cuatro veces la apuesta.
    • Si sale una cereza, se devuelve la apuesta.
      • Si además hay dos frutas iguales, se devuelve tres veces la apuesta.
    • Si salen tres frutas iguales que no sean cerezas, se devuelve cinco veces la apuesta.
    • Si salen dos frutas iguales que no sean cerezas, se devuelve dos veces la apuesta.
    • En el resto de casos, se pierde el dinero.
        if ($a == $cereza && $b == $cereza && $c == $cereza) {
        $premio = 10;
    } elseif (($a == $cereza && $b == $cereza) || ($b == $cereza && $c == $cereza)
        || ($a == $cereza && $c == $cereza)) {
        $premio = 4;
    } elseif ($a == $cereza || $b == $cereza || $c == $cereza) {
        $premio = 1;
        if ($a == $b || $b == $c || $a == $c) {
            $premio = 3;
        }
    } elseif ($a == $b && $b == $c) {
        $premio = 5;
    } elseif ($a == $b || $b == $c || $a == $c) {
        $premio = 2;
    } else {
        $premio = 0;
    }
  • Muestre el premio obtenido (el número de veces que se gana la apuesta) y muestre la cara triste si se pierde y la alegre si se gana. 
    if ($premio == 0) {
    print "          <img src=\"img/face-sad.svg\" alt=\"Mal\" height=\"50px\" />$premio</p>\n";
// Si hemos ganado, cara alegre
} else {
    print "          <img src=\"img/face-smile.svg\" alt=\"Bien\" height=\"50px\" />$premio</p>\n";
}

Complete el programa anterior para que el funcionamiento sea similar al de una máquina tragaperras de verdad

    if ($jugar != "" && $apuesta != 0) {
    $a = rand(1, 6);
    $b = rand(1, 6);
    $c = rand(1, 6);
 ...
  • Al ganar un premio, aumente el número de monedas en posesión del jugador. 
        $apuesta = $apuesta * $premio;
  • Si se meten monedas o si el premio es sólo la devolución de la cantidad apostada, muestre una cara indiferente. 
    if ($premio == 0) {
    print "          <p><img src=\"img/face-sad.svg\" alt=\"Mal\" height=\"80px\" /></p>\n";
} elseif ($premio == 1) {
    print "          <p><img src=\"img/face-plain.svg\" alt=\"Mal\" height=\"80px\" /></p>\n";
} else {

}
