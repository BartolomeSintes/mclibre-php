Google Chart 1 - Convertidor de valores numéricos a cadena (simple encoding)

Escribe un convertidor de valores numéricos bytes a la cadena de valores, que conste de una sola página

  • En la página se solicitan los datos numéricos (inicialmente, cuatro)
  • Se puede aumentar o reducir el número de datos a introducir (entre dos y quince datos)
  • Si los datos introducidos no son todos numéricos, se vuelven a mostrar para ser completados o corregidos
  • Si los datos son correctos, se muestra la cadena correspondiente en formato simple encoding

Ejemplos de resultados (* representa que falta un dato):

  • 10, 40, 20, 30 => A9Up
  • 50, -50, 0, *, 25 => 9Af_u
  • *, 25, 0, 25, 50, -50 => _ufu9A
  • 84, 43, 67, *, 5, 82 => 9dw_A7
  • 1, 1, 1 => fff
  • *, *, *, *, * => _____
  • 5, * => f_

Google Chart 2 - Crea gráfica de líneas

Escribe un formulario que muestre una gráfica a partir de los valores introducidos por el usuario y que conste de una sola página.

  • El formulario de entrada de datos es similar al del ejercicio anterior.
  • El usuario introduce el tamaño de la gráfica.
  • La gráfica se genera mediante Google Chart.
