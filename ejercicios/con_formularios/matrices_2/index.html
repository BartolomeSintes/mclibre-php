Para facilitar la realización de los ejercicios, se proporcionan unas plantillas de los ejercicios Verbos irregulares, que incluyen la página de formulario completa y parte de la página de resultados (los fragmentos HTML).

Se pueden descargar unas posibles soluciones de estos ejercicios en la página de soluciones.

Comentarios

  • La lista de verbos irregulares se encuentra definida como matriz en el archivo irregular_verbs_list.php.
    Para no tener que copiarla y pegarla en cada programa, se puede incluir con include: 
    <?php
include "irregular_verbs_list.php";
?>

    La matriz de verbos tiene dos dimensiones: el primer índice es el número de verbo y el segundo índice indica la forma verbal: 0 = infinitivo, 1 = pasado, 2 = participio, 3 = traducción al español.

    $irregularVerbs:
Array
(
    [0] => Array
        (
            [0] => be
            [1] => was / were
            [2] => been
            [3] => ser
        )

    [1] => Array
        (
            [0] => beat
            [1] => beat
            [2] => beaten
            [3] => golpear
        )
 ...
  • Las formas verbales también están definidas en una matriz en cada programa 
    <?php
$formaVerbalNombre = array("infinitivo", "pasado", "participio");
?>
  • El primer ejercicio sirve de base para los siguientes. En el primer ejercicio se elige al azar un verbo y una forma verbal y se muestra al usuario. En los siguientes ejercicios se muestra al usuario la información incompleta y se pide al usuario que la complete

Escriba un programa de una página que muestre formas verbales de los verbos irregulares ingleses, elegida al azar. Cada vez que se actualice la página, se mostrará una forma verbal distinta.

Verbos irregulares 2

Escriba un programa que conste de dos páginas para practicar los verbos irregulares ingleses:

  • En la primera página se solicita una forma verbal, elegida al azar.
  • En la segunda página se indica si se ha contestado correctamente.

Verbos irregulares 4

Escriba un programa para practicar los verbos irregulares ingleses.

  • Se pide una forma verbal elegida al azar mostrando las tres formas verbales del verbo y se pregunta cuál es la correcta.
  • Si se recibe una respuesta, se indica si es corrrecta o no.
