if ... elseif ... else ... (0) 6 - Convertidor de RGB A HSL

Escriba un programa que convierta códigos de color RGB a códigos de color HSL.

El procedimiento de cálculo (fuente: Wikipedia) es el siguiente:

  • Los valores RGB son enteros de 0 a 255
  • Los valores RGB se dividen por 255 y en todos los cálculos se usan los valores decimales r, g y b
  • M es el máximo de los tres valores
  • m es el mínimo de los tres valores
  • C es M - m
  • Cálculo de H:
    • si C es 0, h es 0 (por convención, realmente está indefinido)
    • si M es r, h es (g-b) / C
    • si M es g, h es (b-r) / C + 2
    • si M es b, h es (r-g) / C + 4
    • si h es negativo, sume 6 a h
    • multiplique h por 60
    • redondee h a un número entero y H es el resto de h entre 360
  • Cálculo de L:
    • L es la media entre M y m
  • Cálculo de S:
    • si L es 1, S es 0
    • si no, S es C / (1 - abs(2*L - 1))
  • redondee L y S con tres decimales
