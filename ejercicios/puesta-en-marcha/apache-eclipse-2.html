Ejercicios - Apache y Eclipse (2)

Apache y Eclipse (2)

Enunciados

  • 1 - Directiva DirectoryIndex
  • 2 - Directiva DocumentRoot

Estos ejercicios tratan de la configuración de Apache y de Eclipse.

Apache 1 - Directiva DirectoryIndex

  • Cree la carpeta C:\tmp\apache-eclipse\e07
  • Cree con NotePad++ dos archivos index.html y home.html diferentes en la carpeta "C:\tmp\apache-eclipse\e07". Por ejemplo: 
    <!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="utf-8" />
  <title>Ejercicio 7</title>
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
</head>
<body>
  <p>Esta es la página index.html del ejercicio 7.</p>
</body>
</html>

    <!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="utf-8" />
  <title>Ejercicio 7</title>
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
</head>
<body>
  <p>Esta es la página home.html del ejercicio 7.</p>
</body>
</html>
  • Cree el alias /ejercicio-07 que enlace a la carpeta "C:\tmp\apache-eclipse\e07"
  • Compruebe que en el navegador se muestra el archivo index.html al escribir http://localhost/ejercicio-07/
  • Modifique en el archivo httpd.conf la directiva DirectoryIndex de manera que se abra primero el archivo home.tml
  • Compruebe que en el navegador se muestra el archivo home.html al escribir http://localhost/ejercicio-07/
  • Deshaga los cambios realizados en el archivo httpd.conf
  • Compruebe que en el navegador se muestra el archivo index.html al escribir http://localhost/ejercicio-07/

Apache 2 - Modificación de DocumentRoot

  • Cree la carpeta C:\tmp\apache-eclipse\e08
  • Cree con Notepad++ en la carpeta C:\tmp\apache-eclipse\e08 un fichero que se llame prueba.html con el siguiente contenido: 
    <!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="utf-8" />
  <title>Ejercicio 8</title>
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
</head>
<body>
  <p>Esta es la página prueba.html del ejercicio 8.</p>
</body>
</html>
  • Modifique en el fichero httpd.conf la directiva DocumentRoot de manera que apunte a la carpeta "C:\tmp\apache-eclipse\e08"
  • Compruebe que se muestra el directorio con el archivo prueba.html al escribir en el navegador la dirección http://localhost
  • Deshaga los cambios realizados en el archivo httpd.conf
  • Compruebe que en el navegador se muestra el panel de administración web de XAMPP al escribir http://localhost
