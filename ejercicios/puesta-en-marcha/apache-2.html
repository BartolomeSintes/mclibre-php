Apache (2)

Enunciados

  • 1 - Directiva DocumentRoot
  • 2 - Modificar DocumentRoot
  • 3 - Directiva DirectoryIndex

Estos ejercicios tratan de la configuración de Apache. Los cambios que se realicen se deben deshacer al terminar los ejercicios, por lo que se recomienda hacer antes una copia de seguridad del fichero de configuración httpd.conf de Apache y restaurarlo al final.

Apache (2) 1 - Directiva DocumentRoot

  • Abra el archivo httpd.conf con el Panel de control de XAMPP.
  • Compruebe que la directiva de Apache DocumentRoot tiene el valor "C:/xampp/htdocs": 
    DocumentRoot "C:/xampp/htdocs"
<Directory "C:/xampp/htdocs">
    #
...
  • En el directorio C:/xampp/htdocs, cree con Notepad++ el fichero e01-1.html. Por ejemplo: 
    <!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="utf-8">
  <title>Ejercicio Apache (2) 1</title>
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>
<body>
  <h1>¡Hola, amigo!</h1>
</body>
</html>
  • Compruebe en el navegador que puede ver la página web escribiendo la dirección http://localhost/e01-1.html:

Apache (2) 2 - Modificar DocumentRoot

  • Cree la carpeta C:\tmp\apache-2\e02
  • Cree con Notepad++ en la carpeta C:\tmp\apache-2\e02 un fichero que se llame prueba.html con el siguiente contenido: 
    <!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="utf-8">
  <title>Ejercicio 8</title>
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>
<body>
  <p>Esta es la página prueba.html del ejercicio 8.</p>
</body>
</html>
  • Modifique en el fichero httpd.conf la directiva DocumentRoot de manera que apunte a la carpeta "C:\tmp\apache-2\e02"
  • Compruebe que se muestra el directorio con el archivo prueba.html al escribir en el navegador la dirección http://localhost
  • Deshaga los cambios realizados en el archivo httpd.conf
  • Compruebe que en el navegador se muestra el panel de administración web de XAMPP al escribir http://localhost

Apache (2) 3 - Directiva DirectoryIndex

  • Cree la carpeta C:\tmp\apache-3\e03
  • Cree con Notepad++ dos archivos index.html y home.html diferentes en la carpeta "C:\tmp\apache-2\e01". Por ejemplo: 
    <!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="utf-8">
  <title>Ejercicio 3</title>
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>
<body>
  <p>Esta es la página index.html del ejercicio 3.</p>
</body>
</html>

    <!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="utf-8">
  <title>Ejercicio 3</title>
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>
<body>
  <p>Esta es la página home.html del ejercicio 3.</p>
</body>
</html>
  • Cree el alias /ejercicio-03 que enlace a la carpeta "C:\tmp\apache-3\e03"
  • Compruebe que en el navegador se muestra el archivo index.html al escribir http://localhost/ejercicio-03/
  • Modifique en el archivo httpd.conf la directiva DirectoryIndex de manera que se abra primero el archivo home.html
  • Compruebe que en el navegador se muestra el archivo home.html al escribir http://localhost/ejercicio-03/
  • Deshaga los cambios realizados en el archivo httpd.conf
  • Compruebe que en el navegador se muestra el archivo index.html al escribir http://localhost/ejercicio-03/
