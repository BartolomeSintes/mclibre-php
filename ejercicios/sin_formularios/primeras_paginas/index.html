Primeras páginas 1 - Hola, mundo

Escriba el típico primer programa que salude al mundo en un único fragmento PHP.

  • No es necesario que la página valide, con que estén las etiquetas <html>, <body> y <p> es suficiente.

Primeras páginas 2 - Hola, mundo (válido)

Escriba el típico primer programa que salude al mundo, que tenga dos fragmentos PHP y dos fragmentos HTML.

  • La página debe ser válida.
  • Hay que crear en Eclipse dos archivos: la página .php y la página .css.
  • La hoja de estilo de estilo se puede copiar del ejercicio de ejemplo: hay que abrir el ejercicio de ejemplo en una pestaña aparte, abrir el código fuente de la página, hacer clic en el enlace a la hoja de estilo y copiar el código css a la página creada.

Enlace a ejercicio Primeras páginas 2

Primeras páginas 3 - Hola mundo en dos páginas

Escriba el típico primer programa que salude al mundo, en dos páginas PHP, que tengan cada una dos fragmentos PHP y dos fragmentos HTML. .

  • La primera página incluye un enlace a la segunda página.
  • La segunda contiene el mensaje "¡Hola, mundo!" y un enlace a la primera página.
  • Las páginas deben ser válidas.
  • La hoja de estilo puede ser la del ejercicio anterior. No hace falta volver a crearla.

Enlace a ejercicio Primeras páginas 3

Primeras páginas 6 - Código de color

Escriba un programa que cada vez que se ejecute muestre un código de color RGB elegido al azar. Un código de color tiene el formato rgb(rojo, verde, azul), donde rojo, verde y azul son números enteros entre 0 y 255.

Primeras páginas 7 - Saludo

Escriba un programa que cada vez que se ejecute muestre un saludo a un tamaño elegido al azar entre 200% y 800%.
