En este ejercicio se debe crear un programa que muestre el número de dados indicado por el usuario. Cada dado tiene un color de fondo rojo o azul al azar. Además, el programa señalará cuántos dados hay del color indicado por el usuario y los mostrará.

Cuenta dados - Comentarios

  • El programa tendrá dos páginas:
    • La primera página muestra un formulario con dos controles:
      • El número de dados que se va a mostrar (entre 2 y 10).
      • El color del que se va a contar y mostrar los dados
    • La segunda página mostrará:
      • Si se han recibido valores incorrectos, los mensajes de error correspondientes.
      • El número de dados indicado por el usuario.
      • Las imágenes de los dados del 1 al 6, al azar, con un color de fondo rojo o azul, también al azar.
      • El número de dados del color indicado por el usuario.
      • Los dados del color indicado por el usuario de entre los mostrados anteriormente.
  • Si se intenta acceder directamente a la segunda página sin pasar por la primera o si se envía a la segunda página información distinta a la esperada, se mostrarán mensajes de error con la información faltante o incorrecta.
  • No se deben cambiar los nombres de los controles incluidos en las plantillas.
  • Al actualizar la segunda página, se mostrarán dados distintos, pero la misma cantidad y se contabilizarán los del mismo color.
