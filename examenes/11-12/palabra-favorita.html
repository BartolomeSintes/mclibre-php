En este examen se debe crear un programa que gestione una base de datos de palabras y que permita votar la palabra favorita.

Mi palabra favorita - Comentarios

  • Para gestionar la base de datos, sólo serán necesarias las opciones: Borrar todo, Votar y Listar.
  • Al añadir una palabra, se almacenará un voto.
  • Cada vez que se vote una palabra, se aumentará en 1 el número de votos.
