En este examen se debe crear un programa que seleccione un número y el usuario tenga que adivinarlo.

El programa tendrá dos páginas:

  • La primera selecciona un número y pide un primer valor al usuario.
  • La segunda dice al usuario si ha acertado o si el número es demasiado alto o bajo y pide un nuevo valor

Adivina el número - Comentarios

  • Las dos páginas son páginas php. El ejemplo utiliza el método get en los formularios para que se vea que se envían los datos.
  • Primera página:
    • Para generar el número a adivinar se pueden utilizar las funciones getrandmax() y rand():
      • getrandmax() devuelve el mayor número que puede generar la función rand().
      • rand($minimo, $maximo) genera un número aleatorio entre $minimo y $maximo.
      $minimo  = 0;
$maximo  = getrandmax();
$secreto = rand($minimo, $maximo);
    • El número a adivinar se puede incluir en un control oculto.
  • Segunda página:
    • Los valores recibidos (tanto el número secreto como el valor escrito por el usuario) deben ser valores numéricos enteros entre 0 y getrandmax().
    • Si el usuario ha adivinado el número, el juego termina.
    • Si el usuario no ha adivinado el número, se le informa de si el valor es demasiado alto o bajo y se vuelve a mostrar un formulario como el de la primera página.
