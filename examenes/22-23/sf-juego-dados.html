En este ejercicio se debe crear un programa que muestre una partida de un sencillo juego de dados.

Juego de dados - Comentarios

  • El programa tendrá una sola página.
  • La página mostrará primero el lanzamiento de un dado por parte de dos jugadores. Esos dados indican el número de dados que lanzará cada jugador a continuación.
  • La página mostrará después los dados lanzados por cada jugador.
  • Ganará el jugador que haya obtenido el valor más alto entre sus dados. Si obtienen el mismo valor más alto, ganará el jugador que haya lanzado menos dados.
  • La imagen del hombre es la entidad numérica &#x1f468; 👨 y la imagen de la mujer es la entidad numérica &#x1f469; 👩
  • Al actualizar la página, se mostrará una nueva partida.
