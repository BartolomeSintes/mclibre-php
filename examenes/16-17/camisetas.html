En este ejercicio se debe crear un programa que permita personalizar una camiseta.

Camisetas - Comentarios

  • El programa tendrá dos páginas:
    • La primera página es un formulario con tres campos:

    • Si no se elige el género o no se escribe texto o el texto tiene más de nueves carácteres, la segunda página mostrará los avisos correspondientes::

      Nota: La longitud de una cadena se puede calcular con la función strlen($cadena), que devuelve la longitud de la cadena.

    • No será necesario realizar ninguna comprobación del código de color.
    • Si no se muestran avisos, se mostrará la camiseta personalizada:
