En este ejercicio se debe crear un programa que muestre una partida de un juego de cartas para dos jugadores.

Juego de cartas - Comentarios

  • El programa debe:
    • Mostrar nueve cartas, al azar, de palos distintos para cada jugador (picas y corazones). Los valores se pueden repetir.
    • Mostrar las cartas ordenadas de mayor a menor y compararlas una a una. Para cada pareja de cartas, el jugador que tiene la carta más grande obtiene dos puntos. En caso de empate, cada jugador obtiene un punto. Gana la partida el jugador que haya obtenido más puntos
    • Mostrar la puntuación de cada jugador e indicar quién ha ganado la partida.
    • Mostrar una partida distinta cada vez que se actualice la página (aunque al generarse al azar puedan repetirse ocasionalmente).
