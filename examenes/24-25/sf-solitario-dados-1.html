En este ejercicio se debe crear un programa que muestre una partida del juego "Solitario de dados".

Solitario de dados - Comentarios

  • Las reglas del solitario son las siguientes:
    • El jugador saca entre cuatro y ocho dados de color rojo o azul, al azar.
    • El jugador separa los dados por colores
    • El jugador comprueba si el valor de algún dado de color azul coincide con el valor de algún dado de color rojo.
  • El programa tendrá una sola página:
    • Se mostrarán todos los dados.
    • Se mostrarán los dados de color azul.
    • Se mostrarán los dados de color rojo.
    • Se indicará si coincide algún dado o no.
  • Al actualizar la página, se mostrará una nueva partida.
