En este ejercicio se debe crear un programa que imite el funcionamiento básico de un solitario de dados.

Tirada de dados - Comentarios

  • El programa podrá tener dos páginas y utilizar sesiones:
    • La primera página muestra tres dados de seis caras:
      • La primera vez que se abre la página se muestran tres dados al azar.
      • Un botón permite volver a tirar los tres dados.
      • En cada tirada se muestra la puntuación más alta alcanzada y la más baja (sumando los tres valores).
      • Un botón permite reiniciar el juego.
    • La segunda página recoge el botón elegido, modifica las variables de sesión y redirige a la primera página.
  • Si se intenta acceder directamente a la segunda página sin pasar por la primera o si se envía a la segunda página información distinta a la esperada, se redirigirá a la primera página sin informar al usuario, ni modificar las cartas ni los contadores.
  • No se debe cambiar el nombre del control incluido en las plantillas.
  • Al actualizar la primera página, no se modificará el contenido (ni los dados ni los valores máximo y mínimo conseguidos).
