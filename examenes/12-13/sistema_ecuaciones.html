En este ejercicio se debe crear un programa que admita los coeficientes de un sistema de dos ecuaciones con dos incógnitas y que muestre las soluciones de la ecuación.

 Sistema de dos ecuaciones - Comentarios

  • El programa tendrá dos páginas:
    • La primera pide los valores.
    • La segunda escribe la respuesta
  • Se deben rellenar todos los campos con valores numéricos.
  • Los sistemas de dos ecuaciones pueden tener infinitas soluciones, una solución o ninguna solución.
    • si a1*b2-a2*b1 y b1*c2-b2*c1 son cero, el sistema tiene infinitas parejas de soluciones x e -a1/b1*x+c1/a1, para cualquier valor de x.
    • si a1*b2-a2*b1 no es cero, el sistema tiene una solución x = (c1*b2-c2*b1)/(a1*b2-a2*b1) e y = (a1*c2-a2*c1)/(a1*b2-a2*b1).
    • si a1*b2-a2*b1 es cero y b1*c2-b2*c1 no es cero, el sistema no tiene solución.
  • Ejemplos de sistemas:
    • 2x+3y = 8 // 3x - 2y = -1. Solución única x = 1 y = 2
    • 2x+3y = 8 // 4x + 6y = 9. No tiene solución
    • 2x+3y = 8 // 4x + 6y = 16. Infinitas soluciones x , -0.66667x+4, para cualquier valor de x
