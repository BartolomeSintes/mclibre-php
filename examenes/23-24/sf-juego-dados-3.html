En este ejercicio se debe crear un programa que simule una partida de un juego de dados de dos jugadores. Cada jugador lanza cinco dados. la puntuación de cada jugador es la suma de los puntos obtenidos en cada dado, descartando los valores máximos y mínimos. Gana la partida el jugador que obtenga más puntos.

Juego de dados (3) - Comentarios

  • El programa debe:
    • Mostrar una partida del juego.
    • Mostrar los 5 dados de cada jugador, al azar.
    • Mostrar la puntuación de cada jugador.
    • Indicar el resultado final de la partida mediante emojis. El ganador se indicará mediante el emoji &#127867; 🍻. En caso de empate, ambos jugadores mostrarán el emoji &#127866; 🍺,
