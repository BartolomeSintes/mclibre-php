En este ejercicio se debe crear un programa que simule la tirada de varios dados (de 1 a 10) de diferentes caras.

Elige dados - Comentarios

  • El programa tendrá dos páginas:
    • La primera página muestra un formulario con dos controles:
      • El número de datos a mostrar (entre 1 y 10).
      • El tipo de dado (sólidos platónicos)
    • La segunda página mostrará:
      • El tipo y número de dados solicitados.
      • El valor de los dados podrá estar girado.
  • Se recuerda que:
    • Un tetraedro tiene cuatro caras triangulares
    • Un cubo tiene seis caras cuadradas
    • Un octaedro tiene ocho caras triangulares
    • Un dodecaedro tiene doce caras pentagonales
    • Un icosaedro tiene veinte caras triangulares
  • Si se intenta acceder directamente a la segunda página sin pasar por la primera o si se envía a la segunda página información distinta a la esperada, se mostrarán mensajes de error con la información faltante o incorrecta.
  • No se deben cambiar los nombres de los controles incluidos en las plantillas.
  • Al actualizar la segunda página, sólo cambiarán los valores de los dados.
