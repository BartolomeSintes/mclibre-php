En este examen se debe crear un programa para indicar cuánta gente ha contestado un encuesta. El programa tendrá tres páginas:

  • La primera pide el número de personas que ha contestado la encuesta y el número de preguntas que tenía la encuesta.
  • La segunda dibuja una tabla de casillas de verificación en la que se indicaría qué preguntas ha contestado cada persona.
  • La tercera cuenta las celdas marcadas por fila.

Encuesta - Comentarios

  • El tamaño de la tabla debe comprobarse (que el número de filas y el de columnas sean números enteros entre el máximo y el mínimo.
  • Para recoger la matriz se debe utilizar una función recogeMatriz2() para recoger una matriz bidimensional, basada en la función recogeMatriz() explicada en la lección de recogida de datos. Por ejemplo: 
    <?php
function recogeMatriz2($var)
{
    $tmpMatriz = [];
    if (isset($_REQUEST[$var]) && is_array($_REQUEST[$var])) {
        foreach ($_REQUEST[$var] as $indice => $fila) {
            $indiceLimpio = trim(htmlspecialchars($indice, ENT_QUOTES, "UTF-8"));
            if (is_array($fila)) {
                foreach ($fila as $indice2 => $valor) {
                    $indice2Limpio = trim(htmlspecialchars($indice2, ENT_QUOTES, "UTF-8"));
                    $valorLimpio   = trim(htmlspecialchars($valor,   ENT_QUOTES, "UTF-8"));
                    $tmpMatriz[$indiceLimpio][$indice2Limpio] = $valorLimpio;
                }
            }
        }
    }
    return $tmpMatriz;
}
?>
  • En la segunda página se añadirá el número de peronas y de preguntas en dos controles ocultos (o con sesiones)
  • En la tercera página no hace falta verificar los valores
