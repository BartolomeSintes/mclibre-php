Exámenes - Punto móvil

  • Comentarios
  • Ejemplo de código HTML

En este ejercicio se debe crear un programa que permita desplazar un punto a derecha e izquierda.

La posición del punto se almacenará en una variable de sesión, no en un control oculto.

Punto móvil - Comentarios

  • El programa tendrá una sola página:
    • Al cargar la página por primera vez, se muestra el punto en el centro de la línea (a 150px):

    • Cada vez que se hace clic en la flecha izquierda, el punto se desplaza 10px a la izquierda.

    • Cada vez que se hace clic en la flecha derecha, el punto se desplaza 10px a la derecha.

    • Cuando el punto llega a un extremo y continúa en la misma dirección, se muestra en el lado opuesto.

Punto móvil - Ejemplo de código HTML

Este código correspondería a la primera captura, con el punto en el centro.
