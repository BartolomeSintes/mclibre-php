En este ejercicio se debe crear un programa que muestre el número de dados indicado por el usuario con el color de fondo indicado por el usuario y con valores generados al azar. Además, el programa mostrará el producto de los valores obtenidos.

Muestra dados - Comentarios

  • El programa tendrá dos páginas:
    • La primera página muestra un formulario con dos controles:
      • El número de dados que se va a mostrar (entre 2 y 15).
      • El color de fondo (rojo, verde, azul y amarillo)
    • La segunda página mostrará:
      • El número de dados indicado por el usuario.
      • Las imágenes de los dados, al azar.
      • El producto de todos los valores.
  • Si se intenta acceder directamente a la segunda página sin pasar por la primera o si se envía a la segunda página información distinta a la esperada, se mostrarán mensajes de error con la información faltante o incorrecta.
  • No se deben cambiar los nombres de los controles incluidos en las plantillas.
  • Al actualizar la segunda página, se mostrarán valores de los dados distintos, pero la misma cantidad y del mismo color.

    <img src="img/3.svg" alt="3" width="100" height="100" style="background-color: yellow;"/>

  <p>El producto de todos los valores es 30.</p>
